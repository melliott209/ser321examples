You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /caesar -- encrypts a String using a Caesar/ROT cipher (ex: /caesar?string=hello&shift=5)
  • /userProjectsByLang -- uses Github API to retrieve the repos by a user written in the given language (ex: /userProjectsByLang?user=melliott209&lang=Python)
